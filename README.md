## 🛠️ Projects Repository

### 1. 🔑 Password Generator (`password_generator.py`)
*   **Description:** A highly secure Python application designed to generate strong, customized random passwords based on user-defined length and complexity preferences.
*   **Key Features:** User-defined length, inclusions of special characters/numbers, and robust input validation.

### 🎮 2. Rock, Paper, Scissors Game (`Rck_Paper_Scissor.py`)
*   **Description:** An interactive digital recreation of the classic childhood game built from scratch.
*   **Key Features:** Smart automated computer moves using Python's `random` module, persistent real-time score tracking, and an infinite command-line game loop with a graceful exit option.

### 📇 3. Contact Book Application (`Contact_book.py`)
*   **Description:** A feature-rich Graphical User Interface (GUI) contact management system that handles complete data management.
*   **Key Features:** 
    *   **Elegant GUI:** Built with Python's `Tkinter` library featuring a dark theme layout.
    *   **Full CRUD Operations:** Seamlessly Add, View, Search, Update, and Delete contacts.
    *   **Data Structure:** Structured using state-management principles to ensure data consistency during runtime operations.
